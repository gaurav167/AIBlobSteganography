# AIBlobSteganography

* A new approach to video steganography based on visually low recepting regions in video frames.
* Uses AI/CV to extract said regions and embed data there.
* Randomises frames and blobs to increase security.



--


## Run tests:
	pytest [<filename>] [<-v>]
from root directory.
